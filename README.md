# Cobalt Raven VideoDownloader
GUI based python video downloader

Advanced Video Downloader
A Python-based GUI tool for downloading videos from YouTube and other supported sites.

🖥️ Clean tkinter GUI — no command line needed
📋 Fetches and displays all available formats/resolutions before downloading
🍪 Cookie support for age-restricted or member-only content (JSON & SQLite formats)
📁 Choose your output directory with a simple folder browser
📊 Real-time progress bar and status updates
🔗 Automatically strips playlist parameters from URLs for single-video downloads
🔄 Auto-retries up to 3 times on failed format fetches
🧵 Threaded downloads — UI stays responsive during download

Requirements: Python 3.x · yt-dlp · ffmpeg
